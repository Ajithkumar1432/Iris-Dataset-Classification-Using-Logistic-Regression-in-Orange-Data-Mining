# Iris Dataset Classification Using Logistic Regression in Orange

## Description
This project demonstrates a supervised machine learning workflow in Orange using the Iris dataset. Logistic Regression is applied for multiclass classification, with data exploration, train-test sampling, performance evaluation, and confusion matrix analysis.

## Tools & Technologies
- Orange Tool
- Logistic Regression
- Dataset (Iris)
